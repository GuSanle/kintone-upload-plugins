# kintone-upload-plugins

通过改造plugin-uploader 上传插件来实现  


## 安装
安装 plugin-uploader 插件  
npm install -g @kintone/plugin-uploader  

如果遇到无法安装可能是因为nodejs版本过低。请先更新nodejs。如果正常安装，请忽略
更新：  
npm install -g n  
n latest  

## 配置及执行：
config.js:  
config 中配置kintone信息  
dir 配置插件目录  

初始化代码  
npm install  

执行  
node index.js
